# CPSC481_Project

8 Piece Puzzle Game

# What is this?

For the project designated to this course, we decided to create an 8 Piece Puzzle game that will allow the user to insert any photo and create a puzzle to solve. To add a small twist to this, we will be creating a hint system that will show the user the next best choice to solving the puzzle. 

# Scoring

The user will try to complete the puzzle with as least moves as possible. After each move, the score on the screen will increase.

# What search/heuristic is being used

We will be using A* to create the hint system for the user. If the user uses a hint, their amount of moves done will increase
